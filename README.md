# 회의 일정 조율 (Meeting Scheduler)

여러 참석자의 일정을 한 번에 조율해 회의 시간·회의실·화상 링크를 자동으로 확정해주는 사내 회의 스케줄링 서비스입니다.

## 핵심 흐름

1. **회의 생성**: 주최자가 참석자(필수/선택)를 지정하고, 회의 예정 기간·회의 시간 길이·안건 마감일을 설정
2. **필수 참석자 응답**: 필수 참석자 전원이 후보 슬롯별로 기피/불가를 응답. 가능은 기본으로 설정되어있음
3. **자동 압축 + 회의 형태 계산**: 필수 참석자 전원 응답 완료 시 자동 계산 후회의 일정 후보들을 추린 후 선택 참석자에게 알림
4. **선택 참석자 응답** → 선택 참석자 전원 응답 완료 시 **최종 회의 일정 자동 확정 + 회의실 자동 매칭** 온라인 회의 있을 시 화상회의 링크 생성
5. 필요 시 **참석 재확인**, **완화요청**(하드 충돌 시 필수 참석자에게 양보 요청), **수동 조정**(기간 연장/회의실 재선택/온라인 전환)까지 이어지는 예외 흐름 지원
6. 회의 후 **회의록 작성**(AI 초안 생성 지원, `ANTHROPIC_API_KEY` 설정 시)

## 기술 스택

- Next.js 14 (App Router) + React 18
- TypeScript
- Prisma 5.22.0 ORM + SQLite
- zod, bcryptjs, jose(세션 JWT)
- Turso(원격 libSQL, SQLite 호환 서버리스 DB)
- Vercel

## 로컬 실행

```bash
npm install
npx prisma migrate dev
npm run seed
npm run dev -- -p 3311
```

`http://localhost:3311/login`에서 로그인합니다.

데이터를 초기화하려면:

```bash
rm prisma/dev.db && npx prisma migrate dev && npm run seed
```

## 데모로 바로 체험하기

로그인 화면의 **"데모버전으로 체험하기"** 버튼을 누르면 비밀번호 입력 없이 아래 김민준 계정으로 즉시 로그인됩니다. 회의 생성부터 참석자 응답, 자동 확정까지 전체 흐름을 빠르게 둘러보고 싶을 때 사용하세요.
