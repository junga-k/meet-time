# 테스트 계정 (로컬 개발 전용)

> `prisma/seed.ts`가 생성하는 6명의 계정 정보입니다. 사내 SSO 연동은 이번 빌드 범위 밖이며, 이 계정들은 "회사가 SSO로 이미 발급해둔 계정"이라는 전제를 시뮬레이션하기 위한 로컬 시드 데이터입니다.
>
> **주의**: 이 파일과 "전원 동일한 임시 비밀번호" 관행은 로컬 개발/테스트 편의를 위한 것입니다. 실제 운영 환경에서는 계정별로 서로 다른 임시 비밀번호를 발급하고, 이 파일 같은 평문 목록을 저장소에 두지 않아야 합니다.

## 로그인 방법

1. `npx prisma migrate dev && npm run seed`로 아래 계정을 생성
2. `/login`에서 이메일 + 임시 비밀번호 `1111`로 로그인
3. 최초 로그인 시 화면14(프로필 설정)·화면15(온보딩)를 거쳐 화면1(회의 목록)로 진입
4. 화면16(내 정보)에서 비밀번호를 변경할 수 있음(선택)
